Instructions

General Info - Collecting black coins hurts your health. You have to defeat all monsters and collect the one red coin in each level to win. The number of gold coins you collect is your score. If you want, you can time your level and compete against friends for the best time. Coins can be a tiebreaker for such a competition. You have a certain amount of health. If you lose all your health, then you will have to replay.

To move forward - w key
To turn left - a key
To move left - q key
To turn right - d key
To move right - e key
To punch - left click
To crouch - c key
To run forward - shift + w key
To turn left while moving forward - w key + a key
To turn right while moving forward - w key + d key
To turn left while running forward - shift + w key + a key
To turn right while running forward - shift + w key + a key
To turn left while crouching and moving forward - w key + a key + c key
To turn right while crouching and moving forward - w key + d key + c key
To turn left while crouching and running forward - shift + w key + a key + c key
To turn right while crouching and running forward - shift + w key + a key + c key
To jump - Space bar
To walk and jump - w key + spacebar
To run and jump - w key + shift + spacebar
